This example shows how to embed SVG elements in X/HTML served with text/html MIME. The code is basically the same as the previous example (in this case fully integrated into the Feature Showcase without iframe).

In theory SVG (or any other non-XHTML element like XUL or MathML) cannot be embedded in X/HTML served with text/html, this is partially true. In practice appropriately reinserting the markup (using a similar technique to this, although the ItsNat approach is better) and avoiding innerHTML to insert SVG, we can use embedded SVG (or XUL or MathML...) with no problem and no special coding, "smart reinsertion" is done automatically by ItsNat when the page is loaded and ItsNat appropriately inserts new SVG markup.

Another interesting teaching of this example is how to dynamically load/insert/remove a <style> element into the <head> of the page.
