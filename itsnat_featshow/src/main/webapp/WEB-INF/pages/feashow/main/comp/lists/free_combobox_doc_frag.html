Any element group able to be an ElementList can be a free combo box. Free combo boxes implement the interface ItsNatFreeComboBox

Basically free combo boxes work the same as the <select> homologue, the main difference is free combo boxes need one child item element used as the list pattern and there is no default decoration of the selected item.

Free combo boxes do not support in-place editing because the data model does not support list item changes.
