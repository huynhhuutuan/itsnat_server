ItsNat uses Batik DOM under the hood as the W3C DOM Level 2 provider, Batik DOM does not support the W3C DOM Style Level 2.

For ItsNat user the most interesting W3C DOM Style interface is the org.w3c.dom.css.ElementCSSInlineStyle because it helps to manage the style attribute decomposing it as property name/value pairs.

Any DOM org.w3c.dom.Element in ItsNat implements the interface org.w3c.dom.css.ElementCSSInlineStyle,

The ElementCSSInlineStyle goes beyond the standard because is write capable: the style attribute can be modified using the CSS related objects, furthermore, is tolerant to direct changes to the style attribute.

Current ElementCSSInlineStyle implementation is not fully complete but is highly usable.
