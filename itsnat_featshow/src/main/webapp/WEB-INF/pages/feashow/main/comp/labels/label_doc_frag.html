An HTML <label> element can be associated to an ItsNatHTMLLabel component.

This example shows several labels with different Java values using different editors. Any editor is instanced with a double click over the label, the default event type.

In place edition can change the normal end user behavior of a <label>. If you do not need the built-in functionality of <label> specified by HTML and you need in-place edition, use another HTML element (div, p etc) and ItsNatFreeLabel.
