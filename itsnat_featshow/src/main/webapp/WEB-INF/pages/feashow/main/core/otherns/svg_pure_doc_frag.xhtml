This example shows how pure SVG documents are first class ItsNat documents like X/HTML including events (AJAX or SCRIPT), Comet, fragments etc. Most of the features of ItsNat provided for X/HTML documents are valid for SVG too.

In spite of W3C defines a specific DOM model for SVG, Batik DOM included in ItsNat does not recognize SVG specifically, any SVG document must be managed as a generic XML document (using DOM Core interfaces).
