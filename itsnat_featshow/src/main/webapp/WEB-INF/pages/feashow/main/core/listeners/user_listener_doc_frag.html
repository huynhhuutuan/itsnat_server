This example shows how to register and use a listener of a user defined event (ItsNatUserEvent). A user defined event can be fired from the client with the JavaScript methods:

document.getItsNatDoc().createUserEvent("userEvtName");

document.getItsNatDoc().dispatchUserEvent(targetElem,userEvt);

A user event is dispatched to any listener registered with the specified node (may be null) and name using the method ItsNatDocument.addUserEventListener.

A user event may transport optionally a variable number or arguments.
