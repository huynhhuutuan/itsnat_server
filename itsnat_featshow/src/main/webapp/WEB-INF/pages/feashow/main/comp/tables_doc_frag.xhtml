ItsNatTable components leverage the DOM utility ElementTable adding a data model, javax.swing.table.TableModel, a row and column selection system and in-place editing similar to labels.

Table components optionally include a header (if present in the markup), the header is managed as a list by a header component associated to the table component implementing ItsNatTableHeader.

The selection system is very similar to javax.swing.JTable using two javax.swing.ListSelectionModel models (to rows and columns) and no selection decoration is defined by default.
