The ElementLabel utility interface is the first example of the ItsNat pattern based utilities. The pattern approach uses a markup fragment as a pattern to create new markup fragments.

An ElementLabel object is created calling an ElementGroupManager.createElementLabel method and is attached to a DOM element, it copies the original element content as the pattern, and removes the content initially (this behavior may be changed), when a Java data object is set to the label the pattern is used to fill again the element content then the Java object is rendered using the associated user defined ElementLabelRenderer (if not provided the default is used). The element content may be removed and added again.

The ElementLabel may be seen as a 0-1 element manager.
