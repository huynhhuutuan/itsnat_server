ElementTableFree objects are similar to ElementListFree but with elements organized as tables again ignoring other node types like text nodes.

An ElementTableFree object is created calling an ElementGroupManager.createElementTableFree method again with master and slave modes to manage rows and cells (in master mode rows and cells only can be added, changed or removed using the utility object, in slave mode W3C DOM methods can be used too).

As the homologue list an ElementTableFree implements java.util.List including iterators, this list must be seen as a list of row elements.
