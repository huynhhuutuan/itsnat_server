ItsNatTree components leverage the DOM utility ElementTree adding a data model, javax.swing.tree.TreeModel, a tree node selection system and in-place editing similar to labels. Almost single tree node is needed as pattern (trees are all "free" because there is no HTML <tree> tag).

The selection system uses javax.swing.tree.TreeSelectionModel objects and no selection decoration is defined by default. The Feature Showcase defines a class FreeTreeDecorator reused by the tree examples (this class is not part of ItsNat Framework use it as you want).

Node expansion and collapsing is supported by the component as a response of mouse events but there is no visual impact by default, the component fires javax.swing.event.TreeExpansionEvent events to registered TreeWillExpandListener calling ItsNatTree.addTreeWillExpandListener and to registered TreeExpansionListener listeners calling ItsNatTree.addTreeExpansionListener listeners if expansion/collapsing was not vetoed. Again the FreeTreeDecorator class is used to expand and collapse tree nodes.

Tree components support rootless and tree-tables and they are supported by the FreeTreeDecorator.
