Any AJAX framework has some kind of "wait" image or message to tell the user a server request is being processed.

Following the ItsNat philosophy, there is no by default visual notification of AJAX/SCRIPT events. ItsNat allows registering a user defined JavaScript listener to be called when a new event is sent to the server and when it returns. This listener must be a JavaScript object with two methods: after and before.

The user defined event monitor is registered with the JavaScript call document.getItsNatDoc().addEventMonitor(monitor), and unregistered with document.getItsNatDoc().removeEventMonitor(monitor).
