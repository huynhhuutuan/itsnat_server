ItsNat supports out of the box a single selection HTML <select> as a combo box component implementing ItsNatHTMLSelectComboBox.

A combo box uses a javax.swing.ComboBoxModel as data model, and supports a single selection, a selection change can be detected registering a java.awt.event.ItemListener listener calling ItsNatComboBox.addItemListener(ItemListener) because combo boxes implement java.awt.ItemSelectable.

Combo boxes do not support in-place editing because the data model does not support list item changes.
