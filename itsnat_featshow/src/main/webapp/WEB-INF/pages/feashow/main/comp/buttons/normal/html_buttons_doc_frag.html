ItsNat supports out the box the HTML "intrinsic" buttons:

  • <input> based (with types button,image,submit and reset) implementing the interfaces ItsNatHTMLInputType.
  • <button> based implementing the interface ItsNatHTMLButton.
  • <a> based implementing the interface ItsNatHTMLAnchor.

This example shows how we can add three type of listeners: DOM listeners (listening the "click" DOM event), ChangeListeners (listening button state changes), and ActionListeners (to detect when the button was clicked).

Submit and reset buttons outside forms behave as normal buttons.
