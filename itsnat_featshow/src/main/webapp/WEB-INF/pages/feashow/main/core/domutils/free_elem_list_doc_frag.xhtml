An ElementListFree object helps to manage the attached markup as an indexed DOM Element list ignoring other nodes like text nodes.

This utility object is created calling an ElementGroupManager.createElementListFree method with two modes, master and slave.

  • In master mode the utility object must the only way to add, change or remove elements of the list (methods like Node.appendChild cannot be used), this mode is the fastest.

  • In slave mode the attached DOM element list can be modified (add, change or remove elements) using W3C DOM methods too, the utility list ever manifests the current DOM subtree state. This mode is the slowest.

An interesting feature is the java.util.List implementation including iterators, java.util.Collections methods such as sort, shuffle or rotate can be used.
