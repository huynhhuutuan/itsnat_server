This is not the first we build a user defined component, this example goes beyond, the new component type is registered in the framework and can be instanced by ItsNat for instance with a call ItsNatComponentManager.createItsNatComponentById.

This feature is achieved registering a user defined component factory/filter implementing CreateItsNatComponentListener and registering it calling ItsNatDocumentTemplate.addCreateItsNatComponentListener. The registered factory/filter object is called when a new component is requested by a call to a normal ItsNat factory method like ItsNatComponentManager.createItsNatComponentById. The factory/filter has the opportunity to create new component types, replace a just created component or reject it.

This example shows how we can register a new component type LoginComponent with name "login" using a registered factory/filter object LoginCreationItsNatComponentListener. This new login component defines a listener type ValidateLoginListener, this listener is called when the login must be validated.
