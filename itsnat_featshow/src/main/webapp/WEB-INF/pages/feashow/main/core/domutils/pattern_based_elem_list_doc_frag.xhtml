The ElementList utility interface is a pattern based element list, unlike an ElementListFree DOM elements of the list have the same tag and are created using a pattern approach. An ElementList object is created calling an ElementGroupManager.createElementList method and attached to a DOM element parent of the list.

The original markup of the subtree being associated to the ElementList must contain one list item element, this will be the pattern and will be saved internally by the list and used to create new list elements.

A Java data object can be rendered to an element list item using a ElementListRenderer if provided (otherwise the default is used). The element provided to render the data object as markup is the element parent of the list item but it can change using a user defined ElementListStructure.
