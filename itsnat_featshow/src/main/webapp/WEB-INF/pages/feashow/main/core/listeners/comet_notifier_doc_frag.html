ItsNat provides support to the COMET (server push, long polling) technique with no special servlet container.

A server long running process can notify the client asynchronously calling CometNotifier.notifyClient(), this call commands ItsNat to send any pending JavaScript code to the client with no user interaction or timer.

Document modifications can be done in any time locking the document or when a Comet event is dispatched to registered listeners, this event is dispatched when the client is going to be notified (a web request is returning to the client).

The CometNotifier object is created with ClientDocument.createCometNotifier().

Finally the ItsNatDocument does not need be syncronized before calling CometNotifier.notifyClient().
