Components can be fully declared on markup and automatically built with a single call to ItsNatComponentManager.buildItsNatComponents(Node),

This method traverses the specified subtree creating the components declared explicitly (with itsnat:compType attribute) or implicitly (a <button> element has an implicit component, ItsNatHTMLButton), registering these component objects into the framework. Later these components can be queried with calls to ItsNatComponentManager.findItsNatComponentById(String) or ItsNatComponentManager.findItsNatComponent(Node).

Form based elements have built-in components and usually do not need extra information.

Automatic component build can be specified globally with methods: ItsNatServletConfig.setAutoBuildComponents(boolean) and DocumentTemplage.setAutoBuildComponents(boolean).
