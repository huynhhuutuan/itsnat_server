This example shows how to use a self-contained custom component in a jar and in the same time how to encapsulate a HTML layout into a single custom tag, for instance this example shows how a simple <login> element is finally converted by the component to conventional HTML with user, password and enter controls, and how single components can be composed to build a more complex one. Java code and HTML layouts are built into a Java jar (more custom components could be included in the same jar).

Component registering is necessary in feashow_servlet initialization, there you can find code like:

docTemplate.addCreateItsNatComponentListener(new LoginTagCreationItsNatComponentListener());

...

LoginTagComponent.registerTemplate(itsNatServlet);
